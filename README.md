## Day 1 ‚Äì Introduction to Programming & Python Basics
- **Introduction to Programming** ‚Äî Programming concepts, why Python for AI/ML, environment setup.
- **Basic Terminologies** ‚Äî Variables, data types, operators, control structures, functions and scope.
- **Introduction to Python** ‚Äî Syntax, indentation, first program, interactive mode vs scripts.
- **Virtual Environments** ‚Äî `venv`, installing packages with pip, managing dependencies.
- **String Manipulation** ‚Äî Methods, formatting, slicing, indexing, regex basics.

---

## Day 2 ‚Äì Operators and Lists
- **Operators** ‚Äî Arithmetic, comparison, logical, bitwise, membership, identity; precedence and associativity.
- **Lists** ‚Äî Creating and indexing, list methods, slicing, comprehensions, nested lists.

---

## Day 3 ‚Äì Tuples, Sets, and Dictionaries
- **Tuples** ‚Äî Immutable sequences, tuple methods, when to use them over lists.
- **Sets** ‚Äî Union, intersection, difference, symmetric difference; adding and removing elements.
- **Dictionaries** ‚Äî Key-value pairs, methods, iteration, nested dictionaries.

---

## Day 4 ‚Äì Conditional Statements and Loops
- **Conditionals** ‚Äî `if`/`elif`/`else`, nested conditions, ternary expressions.
- **Loops** ‚Äî `for` and `while`, `break`/`continue`/`pass`, `range()`, nested loops.

---

## Day 5 ‚Äì Functions and Arrays
- **Functions** ‚Äî Defining and calling, parameters and arguments, return values, scope, lambdas, recursion.
- **Arrays** ‚Äî `range()`, NumPy array introduction, lists vs NumPy arrays, element-wise math.

---

## Day 6 ‚Äì JSON, Regex, and Exception Handling
- **JSON & RegEx** ‚Äî Parsing and serializing with the `json` module; pattern matching with `re` (`match`, `search`, `findall`, `sub`).
- **Try / Except** ‚Äî Exception types, `try`/`except`/`else`/`finally`, `raise`, custom exception classes.

---

## Day 7 ‚Äì Object-Oriented Programming and File Handling
- **OOP** ‚Äî Classes and objects, `__init__` and `self`, instance vs class attributes, the four pillars (encapsulation, inheritance, polymorphism, abstraction), `super()`, MRO, `@property`, abstract base classes.
- **File Handling** ‚Äî `open`/`close`, file modes, reading and writing, context managers (`with`), binary files, file exception handling.

---

## Day 8 ‚Äì Introduction to NumPy
- **NumPy Foundations** (`01_Numpy.ipynb`) ‚Äî Why NumPy beats Python lists, vectorization, array creation (`array`, `zeros`, `ones`, `arange`, `linspace`), `shape`/`dtype`, dtype gotchas.
- **Array Operations** (`02_Numpy.ipynb`) ‚Äî Indexing, slicing, views vs copies, boolean masking, `np.where`, reshaping, broadcasting, stacking and splitting.

---

## Day 9 ‚Äì NumPy Math, Statistics & Linear Algebra
- **Math & Statistics** (`03_Numpy.ipynb`) ‚Äî ufuncs (`sqrt`, `exp`, `log`, trig, rounding), aggregations and the `axis` argument, `corrcoef`, `np.unique`.
- **Linear Algebra** ‚Äî Dot product and the `@` operator, `np.linalg` (`det`, `inv`, `solve`, `eig`), linear regression from scratch via the Normal Equation.

---

## Day 10 ‚Äì Real-World NumPy & The Bridge to ML
- **Real-World NumPy** (`04_Numpy.ipynb`) ‚Äî Loading and saving (`loadtxt`, `save`, `savez`), missing values (`np.nan`, `nanmean`), `datetime64`, `np.random.default_rng()`, performance tips.
- **NumPy as the ML Bridge** (`05_Numpy.ipynb`) ‚Äî How pandas, scikit-learn, PyTorch, and TensorFlow all speak NumPy underneath; images as arrays.

---

## Day 11 ‚Äì Introduction to Pandas
- **Pandas Foundations** (`01_Pandas.ipynb`) ‚Äî Series and DataFrame, `read_csv`, inspection (`head`, `info`, `describe`), the Index object.
- **Selecting and Filtering** (`02_Pandas.ipynb`) ‚Äî `.loc` vs `.iloc`, boolean masks with `&`/`|`/`~`, `.query()`, `isin()`, `between()`, `SettingWithCopyWarning`.
- **Cleaning and Transforming** (`03_Pandas.ipynb`) ‚Äî Missing data (`isna`, `dropna`, `fillna`), `astype`, `rename`, `.apply()` vs `.map()`, vectorized strings, categorical dtype.

---

## Day 12 ‚Äì Pandas Grouping, Joining, and Reshaping
- **Grouping and Joining** (`04_Pandas.ipynb`) ‚Äî `groupby()` split-apply-combine, `.agg()`, named aggregations, `transform()`, `filter()`, `pivot_table`, `crosstab`, `melt`, `pivot`, SQL-style merges, `concat`.

---

## Day 13 ‚Äì Pandas Time Series, Plotting, and Method Chaining
- **Time Series and Plotting** (`05_Pandas.ipynb`) ‚Äî `DatetimeIndex` and date-string slicing, `.resample()`, rolling and expanding windows, plotting with `.plot()`, method chaining with `.assign()` and `.query()`, performance pitfalls of `iterrows()`.

---

## Day 14 ‚Äì Matplotlib Foundations
- **Foundations** (`01_Matplotlib.ipynb`) ‚Äî pyplot vs object-oriented API, the Figure/Axes model, `plt.subplots()` for grids, line/scatter/bar/histogram, titles, labels, legends, saving to PNG/PDF/SVG.

---

## Day 15 ‚Äì Matplotlib Customization and Styling
- **Customization** (`03_Matplotlib.ipynb`) ‚Äî Color formats (named, hex, RGB, cycle), colormaps (sequential / diverging / qualitative), tick formatters (`FuncFormatter`, `PercentFormatter`), log-scale axes, annotations (`ax.annotate`, `axvspan`, `axhline`), style sheets, fonts, removing chart junk.

---

## Day 16 ‚Äì Matplotlib Core Plot Types
- **Core Plot Types** (`02_Matplotlib.ipynb`) ‚Äî Line plots with full styling (color, linestyle, linewidth, markers), scatter plots with 4D encoding via size and color, bar charts (vertical, horizontal, grouped, stacked), histograms for single distributions, box and violin plots for comparing distributions across groups, pie charts (and the sorted-bar alternative), plus a reference table for picking the right plot type.

---

## Day 17 ‚Äì Matplotlib Advanced and Real-World
- **Advanced and Real-World** (`04_Matplotlib.ipynb`) ‚Äî Non-uniform layouts with `GridSpec`, twin y-axes (`ax.twinx()`) for two scales on one chart, heatmaps via `imshow` (activity, correlation, confusion matrices), publication-quality saves (`dpi=300`, `bbox_inches='tight'`, vector formats), seaborn as the higher-level layer for statistical plots, and a complete multi-panel revenue dashboard pulling everything together.

---

- **Seaborn Foundations** (`01_Seaborn.ipynb`) ‚Äî What seaborn adds on top of matplotlib, `sns.set_theme()` for global styling, the `penguins` built-in dataset, the `hue=` argument for coloring by category in one call, long-format vs wide-format data and `pd.melt`, the figure-level vs axes-level function split, and mixing seaborn with matplotlib via the returned axes / FacetGrid.

##  How to Use This Repository

1. **Clone the repository:**
   ```bash
   git clone https://github.com/AyeshaAfridi26/100-Days-of-AI.git
   cd 100-Days-of-AI
   ```

2. **Set up virtual environment (recommended):**
   ```bash
   python -m venv env
   source env/bin/activate  # On Windows: env\Scripts\activate
   ```

3. **Explore the notebooks:**
   ```bash
   jupyter notebook
   ```

4. **Follow along with the daily lessons and exercises**
